# Rendermind Brand Guidelines v1.0

## 1. Identity & Vision
**Name:** Rendermind.io  
**Slogan:** The Infrastructure for Next-Gen Creative Engineering.  
**Mission:** Provide robust, AI-powered graphics infrastructure for platforms that need to scale visual content creation.

## 2. Visual Language
The brand should feel: **Technical, High-Performance, and Sophisticated.**

### 2.1 Color Palette
- **Deep Obsidian (#0B0E14):** Primary Background.
- **Electric Cyan (#00F5FF):** Brand Action / CTA.
- **Accent Slate (#1E2228):** Cards, borders, and UI separators.
- **Tech Silver (#8E949E):** Secondary text and labels.
- **Ghost White (#F1F1F1):** Primary Typography.

### 2.2 Typography
- **Headings:** `Space Grotesk` (Bold) - Futuristic and geometric.
- **Body:** `Inter` (Regular/Medium) - Clean and professional.
- **Code/Technical:** `JetBrains Mono` - Used for micro-copy and technical specs.

### 2.3 UI Elements
- **Borders:** 1px solid `#1E2228`.
- **Backgrounds:** Subtle dot-grid patterns to evoke a "Canvas" feel.
- **Animations:** Fast, linear-out-slow-in transitions (150ms - 300ms).

## 3. Core Services
1. **Custom Creative Builders:** In-app editors (Canvas/WebGL).
2. **Automated Asset Pipelines:** Mass-scale image/graphic generation via API.
3. **AI Visual Orchestration:** Connecting LLMs with complex graphical layouts.
